4.3 Username Enumeration

 Username enumeration is a security vulnerability where attackers identify valid usernames on a server or web application, often facilitated by different server responses for existing and non-existing users. Enabling attackers to use brute-force techniques to either guess or confirm valid users in a system.

7.2 SQL Injection

 SQL injection (SQLi) is a web security vulnerability that lets an attacker make queries with the database by injecting an SQL command via the input from users to the application. Which allows an attacker to extract sensitive information, modify database data, execute administration operations on the database, and in some cases, issue commands to the operating system.
